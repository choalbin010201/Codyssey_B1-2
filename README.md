# Codyssey_B1-2# Mission02. AI 브랜드 광고 영상 제작

## 1. 과제 개요

본 과제는 AI 기반 브랜드 또는 서비스의 홍보 영상을 제작하는 것을 목표로 한다.  
최종 결과물은 **10초 분량의 세로형 AI 브랜드 광고 영상**이다.

본 프로젝트에서는 소상공인을 위한 AI 홍보 포스터 생성 서비스인 **Poster Coach**를 가상의 브랜드로 설정하고, 해당 서비스를 소개하는 짧은 광고 영상을 제작했다.

---

## 2. 브랜드 소개

### 브랜드명

**Poster Coach**

### 서비스 한 줄 소개

소상공인이 가게 정보만 입력하면, AI가 홍보 포스터 문구와 디자인 구성을 빠르게 제안해주는 서비스

### 핵심 가치

- 홍보 문구 작성 부담 감소
- 포스터 제작 시간 단축
- 디자인 경험이 부족한 소상공인도 쉽게 활용 가능
- SNS, 당근마켓, 네이버 플레이스 등 온라인 홍보에 활용 가능

---

## 3. 영상 콘셉트

영상은 홍보 포스터 제작에 어려움을 겪는 카페 사장님이 Poster Coach 서비스를 통해 빠르게 포스터를 완성하고, SNS에 홍보물을 게시하는 흐름으로 구성했다.

전체 메시지는 다음과 같다.

```text
홍보 문구와 포스터 제작이 어려운 소상공인도
Poster Coach를 사용하면 쉽고 빠르게 홍보 포스터를 완성할 수 있다.
```

## 6. 장면별 설명

### Scene 1. 문제 제시

첫 장면에서는 카페 사장님이 노트북 앞에서 홍보 포스터 문구를 고민하는 모습을 보여준다.  
테이블 위에는 포스터 초안, 메모지, 노트북, 커피잔이 놓여 있어 홍보물을 직접 만들기 어려워하는 상황을 표현했다.

### Scene 2. 서비스 등장

두 번째 장면에서는 노트북 화면에 Poster Coach 웹페이지를 보여준다.  
웹페이지에는 AI 포스터 생성 서비스의 메인 화면, 입력창, 이미지 업로드 영역, Generate Poster 버튼, 포스터 템플릿이 나타난다.

### Scene 3. 결과 제시

마지막 장면에서는 스마트폰 화면에 완성된 카페 홍보 포스터가 SNS 게시물 형태로 나타난다.  
배경의 사장님은 만족스러운 표정으로 웃고 있으며, AI 서비스가 문제를 해결했다는 느낌을 전달한다.

```text
warm, bright, soft corporate, cozy cafe mood
```

## 10. 제작 과정

1. Poster Coach라는 AI 홍보 포스터 생성 서비스를 기획했다.
2. 10초 광고에 맞게 3개의 장면으로 스토리보드를 구성했다.
3. 각 장면에 맞는 이미지를 생성했다.
4. Kling AI를 활용하여 이미지 기반 영상을 생성했다.
5. 영상 편집 도구에서 장면 순서를 구성하고 자막과 배경음악을 추가했다.
6. 최종적으로 10초 분량의 세로형 광고 영상으로 완성했다.
